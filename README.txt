В этой программе аргументы командной строки должны вводится следующим образом:

ИмяПрограммы ВходнойФайл.txt ВыходнойФайл.txt -f n для файлового ввода, n - количество строк для считывания в файле.

ИЛИ

ИмяПрограммы ВходнойФайл.txt ВыходнойФайл.txt -r n для системной генерации, n - количеством генерируемых объектов.

n может быть не равно количеству строк в файле. Если n больше, то будет считатан весь файл так, будто n было 
равно числу строк. Если n меньше, то некоторые строки с конца будут не преобразованы в объекты.

Ввод данных о траспортах в входной файл по такому шаблону:

Ship 36 6078 2233 3

Первое слово может быть Ship, Plane или Train - это тип транспорта
Второе - это скорость, целое число.
Третье - это длина пути, действительное число.
Четвертое - некоторый целочисленный параметр - он есть у всех видов транспорта.
Пятое у кораблей - тип корабля. Либо 1, либо 2, либо 3.
Пятое у самолётов - радиус полёта.
Пятого у поездов нет.
